## Weekly Lab Assignments & Learning Objectives

### **Week 1 — Lab Assignment 1**  
**Learning Objective:**  
Model a given problem using a state-space representation and solve it using BFS/DFS.

---

### **Week 2 — Lab Assignment 2**  
**Learning Objective:**  
Design a graph search agent and understand the role of hash tables and queues in state-space search.

---

### **Week 3 — Lab Assignment 3**  
**Learning Objective:**  
Understand heuristic functions to reduce search space size and explore non-classical search algorithms for large problems.

---

### **Week 4 — Lab Assignment 4**  
**Learning Objective:**  
Non-deterministic Search & Simulated Annealing for large problem spaces using randomized search with partial/full domain knowledge.

---

### **Week 5 — Lab Assignment 5**  
**Learning Objective:**  
Learn graphical models for inference under uncertainty, build Bayesian Networks in R, derive structure and CPTs from data, and perform Naive Bayes classification with feature dependencies.

---

### **Week 6 — Lab Assignment 6**  
**Learning Objective:**  
Understand the working of Hopfield Networks and apply them to combinatorial problems.

---

### **Week 7 — Lab Assignment 7**  
**Learning Objective:**  
Learn data structures for state-space search tasks, use random numbers in MDP/RL, and understand exploration–exploitation with n-armed bandits and epsilon-greedy strategies.

---

### **Week 8 — Lab Assignment 8**  
**Learning Objective:**  
Understand sequential decision-making in stochastic environments and its connection to reinforcement learning.
